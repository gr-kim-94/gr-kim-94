7년 넘는 iOS 개발 경험을 바탕으로, 최근에는 RAG와 LangGraph 기반 AI 서비스 개발까지 확장하고 있는 개발자입니다. SwiftUI, Combine, Objective-C, WebRTC 기반 앱 개발부터 레거시 리팩토링, 대규모 앱 운영, 멀티 에이전트 워크플로우 설계, AWS 기반 배포와 성능 개선까지 실서비스 관점에서 문제를 해결해왔습니다.

## 핵심 역량

- iOS 서비스 개발: Swift, SwiftUI, Combine, Objective-C, SwiftData 기반으로 신규 앱 설계부터 배포, 운영까지 수행
- 실시간 기능 구현: WebRTC, WebSocket, MQTT, Unity 연동 환경에서 채팅 및 화상회의 기능 개발
- AI/LLM 서비스 개발: Qdrant 기반 RAG 아키텍처 설계, LangGraph 기반 멀티 에이전트 워크플로우 구축
- 품질 및 운영 개선: Objective-C -> Swift 전환, UnitTest 기반 리팩토링, QA 배포 프로세스 개선 경험
- 협업과 리딩: iOS 파트 리더로 150개 이상 엔터프라이즈 앱 운영 및 유관 부서 협업 경험

## 주요 성과

- SK네트웍스 FAMILY AI 캠프 최종 프로젝트 1위: 방한 외국인을 위한 여행지 추천 서비스를 Multi-Agent RAG 구조로 설계 및 구현하고, TTFT 4~5초 단축 및 전체 레이턴시 20초에서 10초로 개선

- 실서비스 iOS 프로젝트 다수 수행: RFICE 가상오피스, RemoteMeeting 화상회의, 마이홈플러스 등에서 신규 기능 개발과 성능 개선 수행
- 개인 iOS 앱 2종 App Store 출시: 차비서, 앞바르여행 출시 및 운영
